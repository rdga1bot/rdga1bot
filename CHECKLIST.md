# rdga1bot — Чеклист запуску

## Перед першим запуском

### Залежності
- [ ] `g++` з підтримкою C++17: `g++ --version`
- [ ] OpenCV 4.x: `pkg-config --modversion opencv4`
- [ ] X11/XTest/XExt: `pkg-config --exists x11 xtst xext && echo ok`
- [ ] ncurses: `pkg-config --exists ncursesw && echo ok`
- [ ] cmake + ninja: `cmake --version && ninja --version`
- [ ] `curl` для Telegram: `which curl`

### Збірка
```bash
# Рекомендовано (CMake):
cmake --preset debug-fast && ninja -C build_debug

# Або старий спосіб:
bash build.sh
```
- [ ] Збірка завершується без помилок
- [ ] `./rdga1bot` або `./build_debug/rdga1bot` — бінарник існує і не 0 байт

## Налаштування гри

### Macro /target
- [ ] В грі: відкрити редактор макросів (ALT+M або меню)
- [ ] Створити макрос: `/target МобНейм1` → повісити на слот 2
- [ ] Створити макрос: `/target МобНейм2` → повісити на слот 3
- [ ] (і так далі для всіх типів мобів у зоні фарму)
- [ ] Переконатись що MacroKeys у конфігу відповідає слотам (2,3,4,5,6)

### Хотбар
| Слот | Дія | Конфіг |
|------|-----|--------|
| 2-6 | /target макроси | MacroKeys = 2,3,4,5,6 |
| F1 | Основна атака / скіл | AttackKeys = F1 |
| F3 | Spoil (тільки Spoiler) | SpoilKey = F3 |
| F4 | Sweep (тільки Spoiler) | SweepKey = F4 |
| F5 | /pickup (підбір лута) | LootKey = F5 |
| F6 | HP зілля | HP_Key = F6 |
| F7 | MP зілля | MP_Key = F7 |
| F8 | CP зілля | CP_Key = F8 |

### Налаштування гри
- [ ] Вікно гри: windowed mode (не fullscreen)
- [ ] Chase camera mode увімкнено (стрілки ліво/право = поворот)
- [ ] Персонаж залогінений і в зоні фарму

## Запуск

```bash
# Перший запуск — TUI налаштування
./launch.sh

# Наступні рази — швидкий старт
./rdga1bot --quick

# Примусовий TUI (перенастроювання)
./launch.sh --setup
```

### При запуску перевірити:
- [ ] Debug вікно відкрилось (показує кадр гри)
- [ ] FPS у кутку: 30+
- [ ] Стан в overlay: `State: IDLE` або `TARGETING`
- [ ] HP/MP/CP значення відображаються коректно (не 0 постійно)

## Діагностика

### HP/MP/CP не детектується (завжди 0)
```bash
# 1. В грі натиснути F12 → збереже calibrate_*.png
# 2. Переглянути файли:
#    calibrate.png        — повний кадр
#    calibrate_hp.png     — HP бар регіон
#    calibrate_mp.png     — MP бар регіон
# 3. Якщо HP/MP не знайдено — Eyes кольори не відповідають клієнту
```

### Бот не знаходить мобів
- Перевірити правильність назв мобів у макросах `/target`
- Назва чутлива до регістру та пробілів
- Спробувати натиснути макрос вручну — чи таргетується моб?

### Персонаж відкриває чат замість руху
- НЕ повинно траплятись: бот використовує стрілки ↑↓←→, не WASD
- Якщо трапляється — перевірити Hands.h WalkForward() → має бути KeyboardKey::Up

## Клавіші під час роботи

| Клавіша | Дія |
|---------|-----|
| ESC | Зупинити бот |
| PrintScreen | Зберегти `shot.png` |
| Space | Скинути пошук HP/MP/CP барів |
| F12 | Зберегти `calibrate_*.png` для діагностики |

## Критерії успішної роботи

- [ ] FPS 30+
- [ ] `[STATE] IDLE -> TARGETING` з'являється в консолі
- [ ] `[STATE] TARGETING -> ATTACKING` після знаходження моба
- [ ] HP цілі видно в overlay (`Target: XX%`) і воно падає
- [ ] `[STATE] ATTACKING -> LOOTING` після смерті моба
- [ ] `[LOOTING] Вбивство #1` в консолі
- [ ] Персонаж підбирає лут (F5 натискається)
- [ ] Цикл повторюється
